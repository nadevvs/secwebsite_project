# Secure Personal Website & Email Infrastructure

## Overview
This project documents the design, implementation, and operation of a secure and
lightweight personal website with a custom domain email, built using industry
best practices and project management principles.

## Objectives
- Deploy a production-ready personal website
- Minimize attack surface using static-first design
- Implement responsible disclosure (security.txt, PGP)
- Apply structured project management and risk control

## Scope
**In scope**
- Domain & DNS
- VPS-hosted website
- Managed email
- Security hardening
- Monitoring & documentation
